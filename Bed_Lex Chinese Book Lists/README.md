Scripts to generate custom booklists featuring Chinese Language titles in the Bedford and Lexington library collection.  The key need for the scripts was to primarily present the Chinese character forms of the title and author fields when they are present in the 880 fields of a MARC record.
These reports are run on demand instead of being automated like most within this Repo

Execution Plan
* Run query to generate stats for each report
* Compile query results into an Excel File
