Scripts used to generate a monthly report of library of things checkouts at each Cambridge location
Report is produced as Excel files, which are emailed to staff to review prior to distributing to staff throughout member libraries.

Execution Plan:
* Run query to generate checkout totals for each stat group and location for each day of the prior month
* Compile query results into an Excel File, one tab per location
* Email file to designated staff member(s) as an attachment
